# Music Player App 🎧

## Overview

The **Music Player App** is a React-based web application that lets users play, pause, and navigate through their favorite songs. It is designed with a simple and intuitive interface to provide an easy-to-use music listening experience. Users can browse a playlist, select tracks, and control playback seamlessly.

## Features

- Display a list of songs  
- Play and pause audio  
- Skip to next or previous track  
- Show current song title and playback status  
- Responsive and user-friendly interface  
- Built with reusable React components

## Technologies Used

- **React** – For building the user interface  
- **JavaScript (ES6+)** – For app logic and state management  
- **HTML & CSS** – For layout and styling
